Abbreviations Used

reg
Any of the 8-Bit registers A,B,C,D,E,H,L.
dreg
sreg
reg Used as Destination or Source in a move.
byte
Any 8-bit value.
word
Any 16-bit value.
PC
Program Counter.
SP
Stack Pointer.
RM
Register A,B,C,D,E,H,L or memory M pointed by HL.
BD
Either register pair BC or DE (B=BC, D=DE).
rp
Any Register Pair (B=BC, D=DE, H=HL, SP=SP).
drp
srp
rp used as Destination or Source in a move
PSW
Program Status Word: A pseudo-Register Pair used by PUSH and POP to refer to the A accumulator and Status Flags as a single 16-bit unit.
Addr
A 16-Byte address.
label
Name for an address.
b[x]
The contents of the byte at address x.
w[x]
The contents of the bytes at address x and x+1.
port
An I/O port specified by an number from 00h to FFh.
n
A number from 0 to 7 used by RST to calculate an address.

Status Flags

The status flags are treated as an 8-bit register for PUSH/POP PSW.

7 6 5 4 3 2 1 0

S

Z

K

A

-

P

V

C

The flags are:

BitFlagDescription
0 Cf Carry flag; set to carry out of bit 7 in result
1(Vf) Undocumented signed overflow flag^* (8085 only, on 8080 bit is always 1 in PSW)
2 Pf Parity flag; set if the number of bits in the result is even, and reset if it's odd
3 - No flag, bit is always 0 in PSW
4(Af) Auxiliary Carry (A[C]) flag; set to carry out of bit 3 in result
5(Kf) Undocumented signed Underflow Indicator (UI/X5) and comparison flag^* (8085 only, on 8080 bit is always 0 in PSW)
6 Zf Zero flag; set if the result was zero, reset otherwise
7 Sf Sign flag; set to bit 7 of result
